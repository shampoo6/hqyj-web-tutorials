# 浮动
## 什么是浮动？
使元素从文档结构中浮出（漂浮出来），摆脱文档结构（成为独立浮空单位）

## 作用
浮动主要用于一批元素的页面布局

## 用法
在元素上加入float属性就可以浮动了
```css
.item {
    /* float属性，两个备选项：left，right */
    float: left; 
}
```

## 浮动的特点
- 浮动会漂浮出文档，后续元素将填充到当前浮动元素的位置上来。
- 浮动的元素前面，如果还有块级元素，那么浮动出来的元素距离窗口顶部的距离，保持不变
- 当多个元素向相同方向浮动时，文档中上面一个元素会放在下面一个一个元素的前面。

## 应用场景
局部的一批元素需要进行横向排列时，可以考虑使用浮动
