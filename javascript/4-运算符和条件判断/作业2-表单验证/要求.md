# 要求
表单验证(form validation)

- 制作一个表单，包含以下表单项
    - 账号
    - 密码
    - 是否同意《用户协议》的勾选框
- 点击提交按钮后，对表单中的数据进行验证，规则如下：
    - 账号：必填项，不能为空
    - 密码：长度必须在6~10位
    - 是否同意《用户协议》的勾选框：必须勾选
- 任一表单项验证失败时，即为提交失败，弹窗提示失败原因
- 表单项都验证通过，则为提交成功，弹窗提示提交成功
